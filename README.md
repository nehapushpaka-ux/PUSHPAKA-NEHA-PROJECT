# PUSHPAKA-NEHA-PROJECT# 
HR Analytics: Employee Attrition Prediction

## Overview

This project predicts employee attrition using Machine Learning and presents HR insights through an interactive Power BI dashboard. It helps organizations identify employees at risk of leaving and supports data-driven retention strategies.

## Objectives

- Analyze employee attrition patterns
- Build a Decision Tree prediction model
- Evaluate model performance
- Create an interactive Power BI dashboard
- Provide business recommendations

## Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Power BI

## Dataset

- IBM HR Analytics Employee Attrition & Performance Dataset
- Source: Kaggle
- Records: 1,470
- Features: 35

## Dashboard Highlights

- Total Employees
- Employees Left
- Attrition Rate
- Average Monthly Income
- Attrition by Department
- Attrition by Gender
- Attrition by Job Role
- Machine Learning Insights

## Results

- Decision Tree Classifier
- Employee Attrition Prediction
- Interactive Power BI Dashboard
- Business Recommendations for Employee Retention
